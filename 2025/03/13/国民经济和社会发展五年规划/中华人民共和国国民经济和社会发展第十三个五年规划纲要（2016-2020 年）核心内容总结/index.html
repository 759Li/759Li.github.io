国民经济和社会发展五年规划

中华人民共和国国民经济和社会发展第十三个五年规划纲要（2016-2020）核心内容总结

一、核心指导思想与发展目标

（一）指导思想

以科学发展为主题，以加快转变经济发展方式为主线，牢固树立和贯彻落实创新、协调、绿色、开放、共享五大发展理念，推进供给侧结构性改革，统筹推进经济、政治、文化、社会、生态文明建设和党的建设，确保如期全面建成小康社会。

（二）主要目标

  1. 经济发展：2020年GDP和城乡居民人均收入比2010年翻一番，GDP年均增长6.5%；全员劳动生产率年均增长6.6%；常住人口城镇化率达60%，服务业增加值占比达56%。
  2. 创新驱动：研发经费占GDP比重达2.5%，每万人口发明专利拥有量达12件；科技进步贡献率达60%，互联网普及率大幅提升。
  3. 民生改善：现行标准下农村贫困人口全部脱贫，贫困县全部摘帽；城镇新增就业累计超5000万人；基本养老保险参保率达90%；人均预期寿命提高1岁；建设保障性安居工程2000万套。
  4. 生态环境：单位GDP能耗降低15%，二氧化碳排放降低18%；主要污染物排放总量减少（化学需氧量、氨氮降10%，二氧化硫、氮氧化物降15%）；森林覆盖率达23.04%，地级及以上城市空气质量优良天数比率超80%。
  5. 制度完善：各领域基础性制度体系基本形成，法治政府基本建成，开放型经济新体制基本形成。

二、重点领域发展部署

（一）创新驱动发展战略

  1. 强化科技创新引领，突破航空发动机、量子通信等战略前沿领域核心技术，布局国家实验室和重大科技基础设施。
  2. 深入推进大众创业万众创新，建设公共服务平台，推广众创、众包、众扶、众筹模式。
  3. 构建激励创新的体制机制，深化科技管理体制改革，完善成果转化和收益分配机制，健全普惠性创新支持政策。
  4. 实施人才优先发展战略，培育创新型科技人才和各类专业人才队伍，优化人才发展环境。

（二）构建现代产业体系

  1. 工业：实施制造强国战略，提升工业基础能力，发展智能制造和绿色制造；培育节能环保、新一代信息技术等战略性新兴产业，增加值占GDP比重达15%；推动能源生产和利用方式变革，建设清洁低碳现代能源体系。
  2. 服务业：推动生产性服务业专业化和生活性服务业高品质发展，放宽服务业市场准入，扩大对外开放。
  3. 网络经济：构建高速泛在信息网络，发展互联网产业体系，实施国家大数据战略，强化信息安全保障。

（三）区域协调与新型城镇化

  1. 实施区域总体战略，推进西部大开发、东北振兴、中部崛起、东部率先发展，支持革命老区、民族地区、边疆地区发展。
  2. 重点推动京津冀协同发展、长江经济带发展，拓展蓝色经济空间。
  3. 推进新型城镇化，加快农业转移人口市民化，优化城镇化布局，建设和谐宜居城市，健全住房供应体系，推动城乡协调发展。

（四）生态文明建设

  1. 加快建设主体功能区，优化国土空间开发格局，健全配套政策体系。
  2. 推进资源节约集约利用，实施能源、水、土地等总量和强度双控，发展循环经济。
  3. 加大环境综合治理力度，深入实施污染防治行动计划，严密防控环境风险，加强环境基础设施建设。
  4. 加强生态保护修复，开展大规模国土绿化行动，推进重点区域生态修复，维护生物多样性。
  5. 积极应对全球气候变化，控制温室气体排放，增强适应能力，参与全球气候治理。

（五）全方位对外开放

  1. 完善对外开放战略布局，推进国际产能和装备制造合作，加快对外贸易优化升级，提升利用外资和对外投资水平。
  2. 健全对外开放新体制，营造优良营商环境，扩大金融业双向开放，强化服务保障。
  3. 推进“一带一路”建设，健全合作机制，畅通经济走廊，深化人文交流。
  4. 积极参与全球经济治理，维护多边贸易体制，强化区域和双边自由贸易体制建设。

（六）民生保障与社会建设

  1. 脱贫攻坚：推进精准扶贫精准脱贫，创新扶贫方式，支持贫困地区加快发展，完善支撑体系，确保贫困人口全部脱贫。
  2. 教育与健康：推进教育现代化，均衡发展基本公共教育，推进职业教育产教融合，提升高等教育质量；推进健康中国建设，深化医药卫生体制改革，健全医疗保障体系，加强重大疾病防治，广泛开展全民健身。
  3. 民生保障：增加公共服务供给，实施就业优先战略，缩小收入差距，改革完善社会保障制度，积极应对人口老龄化，保障妇女、未成年人和残疾人权益。
  4. 社会治理：完善社会治理体系，增强社区服务功能，发挥社会组织作用，完善社会信用体系，健全公共安全体系，建立国家安全体系。

（七）精神文明与民主法治建设

  1. 加强社会主义精神文明建设，培育和践行社会主义核心价值观，传承优秀传统文化，丰富文化产品和服务，提高文化开放水平。
  2. 加强社会主义民主法治建设，发展社会主义民主政治，全面推进法治中国建设，加强党风廉政建设和反腐败斗争。

（八）军民融合发展

  1. 全面推进国防和军队现代化，加强军队革命化、现代化、正规化建设，完成国防和军队改革目标任务。
  2. 推进军民深度融合发展，完善体制机制，深化国防科技工业改革，实施军民融合发展工程，加强国防动员和全民国防教育。

三、规划实施保障

  1. 发挥党的领导核心作用，加强领导班子和干部队伍建设，凝聚全社会共识和力量。
  2. 完善规划实施机制，加强统筹协调，开展动态监测和评估，明确责任主体和实施进度。
  3. 强化财力保障，优化财政支出结构，保障规划实施的资金需求。
  4. 充分调动全社会积极性，依靠市场主体自主行为实现预期性目标，激发各类主体活力和创造力。
