国民经济和社会发展五年规划

中华人民共和国国民经济和社会发展第十个五年计划纲要（2001-2005）核心内容总结

一、核心指导方针与主要目标

（一）指导方针

  1. 坚持以发展为主题，抓住机遇加快经济增长。
  2. 坚持以结构调整为主线，推进产业、地区、城乡结构优化升级。
  3. 坚持以改革开放和科技进步为动力，深化体制创新与技术创新。
  4. 坚持以提高人民生活水平为根本出发点，增加居民收入，扩大就业。
  5. 坚持经济社会协调发展，推进精神文明、民主法制建设与可持续发展。

（二）主要目标

  1. 宏观经济：GDP年均增长7%，2005年达12.5万亿元；城镇新增就业4000万人，失业率控制在5%左右；价格总水平基本稳定，国际收支平衡。
  2. 经济结构：三次产业增加值比重调整为13%、51%、36%；信息化水平显著提高，地区差距扩大趋势得到控制，城镇化水平提升。
  3. 科技教育：研发经费占GDP比重达1.5%；基本普及九年义务教育，高中和高等教育毛入学率分别达60%、15%左右。
  4. 可持续发展：人口控制在13.3亿以内，森林覆盖率达18.2%，主要污染物排放总量减少10%。
  5. 人民生活：城乡居民收入年均增长5%左右，城镇居民人均住宅建筑面积达22平方米，公共服务进一步完善。

二、经济结构战略性调整

（一）产业结构优化

  1. 农业：稳定粮食生产能力，拓宽农民增收渠道，调整农村经济结构，推进农业产业化经营，加强农村基础设施建设。
  2. 工业：改造传统产业，发展高新技术产业，优化企业组织结构，推进老工业基地调整，以信息化带动工业化。
  3. 服务业：发展生活性与生产性服务业，推进市场化、产业化、社会化，打破垄断，放宽准入。
  4. 信息化：广泛应用信息技术，建设信息基础设施，发展电子信息产品制造业。

（二）区域与城乡协调发展

  1. 西部大开发：加强基础设施与生态建设，发展特色产业，加大政策支持力度。
  2. 中部崛起：发挥区位与资源优势，巩固农业基础，发展制造业与高新技术产业。
  3. 东部提升：率先体制创新与科技进步，辐射带动中西部发展。
  4. 城镇化：形成大中小城市协调发展的城镇体系，重点发展小城镇，消除城乡体制障碍。

（三）基础设施建设

  1. 水利：加强大江大河治理，推进南水北调工程，解决水资源短缺问题。
  2. 交通：建设“五纵七横”公路网、“八纵八横”铁路网，完善沿海港口与民航机场布局。
  3. 能源：优化能源结构，推进西电东送、西气东输，建立石油战略储备。

三、科技、教育与人才战略

（一）科技进步与创新

  1. 为产业升级提供技术支撑，突破关键共性技术。
  2. 推进高技术研究，在信息、生物、新材料等领域实现突破。
  3. 加强基础研究，建设国家创新体系，深化科技体制改革。

（二）教育发展

  1. 巩固九年义务教育，扩大高中与高等教育规模。
  2. 大力发展职业教育与终身教育，深化教育体制改革，增加教育投入。

（三）人才战略

  1. 培养高素质领导、专业、企业家、公务员与技术工人队伍。
  2. 营造人才成长环境，完善激励机制，促进人才合理流动。

四、人口、资源与环境可持续发展

（一）人口与生育

  1. 稳定低生育水平，人口自然增长率控制在9‰以内。
  2. 提高出生人口素质，加强生殖健康服务。

（二）资源节约与保护

  1. 节约水资源，提高用水效率，合理开发土地、森林、矿产资源。
  2. 推进资源综合利用，发展循环经济。

（三）生态建设与环境保护

  1. 实施天然林保护、退耕还林还草等生态工程。
  2. 治理水污染、大气污染，提高城市污水与垃圾处理率，强化环境监管。

五、改革开放深化

（一）体制改革

  1. 国企改革：完善现代企业制度，推进产权多元化，健全法人治理结构。
  2. 所有制结构：优化国有经济布局，支持非公有制经济发展。
  3. 市场体系：培育要素市场，打破垄断与地方保护，完善价格机制。
  4. 金融、投资、财税改革：深化国有商业银行改革，确立企业投资主体地位，推进税费改革与分税制完善。

（二）对外开放

  1. 提高开放水平，逐步开放服务领域，适应加入WTO要求。
  2. 优化外贸结构，扩大高新技术产品出口，2005年进出口总额达6800亿美元。
  3. 合理利用外资，引导外资投向中西部与高新技术产业。
  4. 实施“走出去”战略，鼓励企业对外投资与合作。

六、人民生活与社会建设

（一）就业与社保

  1. 扩大就业渠道，发展劳动密集型产业与服务业。
  2. 健全养老保险、医疗保险、失业保险制度，完善社会保障体系。

（二）收入与消费

  1. 深化收入分配改革，调节过高收入，增加中低收入群体收入。
  2. 拓宽消费领域，改善居住与出行条件，扩大服务消费。

（三）社会事业

  1. 发展卫生事业，完善城乡医疗服务体系，推进社区建设。
  2. 加强精神文明建设，弘扬共同理想，繁荣文化事业。
  3. 推进民主法制建设，完善人大与政协职能，依法行政，加强社会治安综合治理。

七、国防建设

加强军队革命化、现代化、正规化建设，推进国防科技工业改革，增强国防实力，维护国家主权与统一。

八、规划实施保障

改善宏观调控，综合运用财政、货币、价格政策；创新实施机制，推进体制改革，制定配套政策，整治市场秩序，提供公共服务，集中力量办好重大工程。
