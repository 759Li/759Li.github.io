国民经济和社会发展五年规划

中华人民共和国国民经济和社会发展第十一个五年规划纲要（2006-2010）核心内容总结

一、核心指导原则与发展目标

（一）指导原则

以科学发展观统领全局，坚持经济平稳较快发展、加快转变增长方式、提高自主创新能力、促进城乡区域协调、加强和谐社会建设、深化改革开放六大原则，推动经济社会全面协调可持续发展。

（二）主要目标

  1. 宏观经济：GDP年均增长7.5%，2010年人均GDP比2000年翻一番；城镇新增就业4500万人，失业率控制在5%以内；价格总水平稳定，国际收支基本平衡。
  2. 结构升级：服务业增加值占比提高到43.3%，研发经费占GDP比重达2%，城镇化率提升至47%。
  3. 资源环境：单位GDP能耗降低20%，主要污染物排放总量减少10%，森林覆盖率达20%，耕地保有量稳定在1.2亿公顷。
  4. 社会民生：国民平均受教育年限达9年，城镇基本养老保险覆盖2.23亿人，新型农村合作医疗覆盖率超80%；城乡居民收入年均增长5%。

二、重点领域发展部署

（一）建设社会主义新农村

  1. 发展现代农业，稳定粮食综合生产能力，推进农业结构调整，健全农业服务与流通体系。
  2. 拓宽农民增收渠道，挖掘农业增值潜力，增加非农产业收入，完善增收减负政策。
  3. 改善农村面貌，加强基础设施建设、环境保护、卫生事业与社会保障，培养新型农民，加大农村投入，深化农村改革。

（二）工业结构优化升级

  1. 加快高技术产业发展，提升电子信息、生物、航空航天、新材料产业竞争力。
  2. 振兴装备制造业，突破重大技术装备、汽车工业、船舶工业核心技术。
  3. 优化能源工业布局，有序发展煤炭、积极发展电力、加快油气开发、大力推进可再生能源利用。
  4. 调整原材料工业结构，优化冶金、化工、建材产业布局，提升轻纺工业水平，推进信息化与工业化融合。

（三）加快发展服务业

  1. 拓展生产性服务业，优先发展交通运输、现代物流、金融服务、信息服务、商务服务。
  2. 丰富消费性服务业，提升商贸、房地产、旅游、市政公用、社区服务、体育产业质量。
  3. 破除行业垄断，放宽准入，完善支持政策，推动服务业市场化、产业化、社会化。

（四）促进区域协调发展

  1. 实施区域总体战略，推进西部大开发、振兴东北老工业基地、促进中部崛起、鼓励东部率先发展，支持革命老区、民族地区、边疆地区发展。
  2. 推进主体功能区建设，划分优化开发、重点开发、限制开发、禁止开发区域，实行分类管理政策。
  3. 促进城镇化健康发展，分类引导人口城镇化，形成合理城镇空间格局，加强城市规划建设管理，健全城镇化体制机制。

（五）建设资源节约型、环境友好型社会

  1. 发展循环经济，推进能源、水、土地、材料节约，加强资源综合利用，强化节约政策措施。
  2. 保护修复自然生态，实施天然林保护、退耕还林还草等生态工程，建立生态补偿机制。
  3. 加大环境保护力度，重点治理水污染、大气污染、固体废物污染，强化环保监管与执法。
  4. 强化资源管理，加强水、土地、矿产、海洋、气候资源的合理开发与保护。

（六）实施科教兴国与人才强国战略

  1. 推进科技自主创新，加强基础研究与前沿技术攻关，强化企业创新主体地位，加大知识产权保护，深化科技体制改革。
  2. 优先发展教育，普及巩固义务教育，大力发展职业教育，提高高等教育质量，加大教育投入，深化教育体制改革。
  3. 壮大人才队伍，建设高素质党政、企业经营管理、专业技术、高技能人才队伍，创新人才工作机制。

（七）深化体制改革

  1. 推进行政管理体制改革，转变政府职能，健全决策机制，深化投资体制改革。
  2. 完善基本经济制度，深化国企改革，健全国有资产监管，推进垄断行业改革，鼓励非公有制经济发展。
  3. 推进财税金融改革，完善财政体制与税收制度，深化金融企业、资本市场改革，健全金融调控与监管机制。
  4. 完善现代市场体系，健全统一开放市场，完善价格形成机制，规范市场秩序。

（八）扩大对外开放

  1. 转变外贸增长方式，优化出口结构，扩大进口，发展服务贸易，完善公平贸易政策。
  2. 提高利用外资质量，引导外资投向高技术、现代服务、中西部等领域，促进利用外资方式多样化。
  3. 实施“走出去”战略，支持企业对外投资与跨国经营，推进国际区域经济合作。

（九）推进和谐社会建设

  1. 做好人口工作，稳定低生育水平，改善出生人口素质，应对人口老龄化，保障妇女儿童与残疾人权益。
  2. 提高人民生活水平，扩大就业，调节收入分配，健全社会保障体系，加大扶贫力度，扩大居民消费。
  3. 提高人民健康水平，完善公共卫生与医疗服务体系，加强疾病防治，深化医疗卫生体制改革。
  4. 加强公共安全建设，增强防灾减灾能力，提高安全生产水平，保障饮食用药安全，维护国家安全与社会稳定，强化应急体系建设。
  5. 完善社会管理体制，加强基层自治组织建设，规范民间组织发展，正确处理人民内部矛盾。

（十）加强民主政治、文化建设与国防军队建设

  1. 发展社会主义民主，完善民主制度，推进法制建设，加强廉政建设。
  2. 繁荣社会主义文化，加强思想道德建设，丰富群众精神文化生活，深化文化体制改革。
  3. 加强国防军队建设，推进军队革命化现代化正规化建设，优化国防科技工业，增强国防动员能力。

三、规划实施保障

建立分类指导实施机制，区分市场主导、政府引导、政府保障等任务类型；调整完善财政、税收、投资、产业等政策；健全规划管理体制，强化约束性指标考核与监督评估。
